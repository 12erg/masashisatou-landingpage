Project overview

プロジェクト概要

Enjoy！プログラミングサークル プログラミングに興味がある方、既に学んでいる方、 WEBの仕事に興味がある方などは、 ぜひ一度我々のサークルに参加してみてください！ 現役エンジニアによって創立された、 チームみんなで切磋琢磨しながらプログラミングスキルを学べる サークルです。 楽しむ × 学べる × 汎用性に特化した学習システムです。

Concept

プロジェクトテーマ

どんな挑戦もできる可能性が０％なんてことはありません。 しかし、最初の１％は自分で掘り起こさなければいけません。 すなわち『行動』することです。 自分から行動することで物事は全て動き始めます。 とはいえ、「プログラミングを学ぶにはまず何からやったら いいんだろう？」、「独学は難しそう、、」、「スクールは料金が 高くてなかなか踏み出せない。」、「学ぶ時間がない。」 こんな声が現実だと思います。 そんな問題を全て解決するべく誕生したのがこのサークルです。 今までのプログラミング学習にはない仕組みがここにはあります！ 日常の１％の時間を学びに使い諦めかけてた未来を 一緒に作りましょう！

Activitie

活動内容

HTMLテンプレート開発

→BootstrapをベースにHTML,CSS,Javascriptを使ったテンプレートを開発、配布しています。

Wordpressテーマ開発

→BootstrapをベースにしたオリジナルWordpressテーマを開発、配布しています。

WebDesign

→ウェブサイトやアプリを作るためにロゴやアイコン等デザインツールを使用したクリエイティブ作成。

勉強会

→週に１回ほどオンラインでメンター主催による自由参加の勉強会しています。(不定期開催)

コミュニティ

→サークルメンバー同士で教え合ったり、自分の今やっている事だったりを楽しく会話できる環境。

Need for you

あなたに必要なもの

  • ✔︎PC

  • コードが書けて、インターネットが使えるものであればなんでもOK!

  • ✔︎デザインツール(無料ツール可)

  • 無料のXD、Adobe Illustrator,Photoshotに加え、Affinity Designer等を使用。

  • ✔︎Discordアカウント

  • ミーティングや普段のコミュニケーションはDiscordで基本的には行います。

  • ✔︎継続力

  • とにかく、毎日1%(15分)だけでも毎日継続することに意味があります。楽しく続けていきましょう！

Let's together!

独学はモチベーションを保ったり、 難しい壁が立ちはだかったとき に挫折もしやすい。しかし、 プログラミングはやはり 最終的には一人で高め続けなければ いけないものでもあると思います。 だからこそ我々の環境は一匹狼が 集まりお互いを高め合える場所なのです。冒頭でも言いました、 1%の行動で未来は変わると。 まずは体験会にてプログラミングや １％Projectを体感してみてください！
